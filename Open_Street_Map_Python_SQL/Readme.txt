This project is part of the Data Analyst Nanodegree Program at Udacity.
The code was written in Python and I also used SQL. The data is from the OpenStreetMap and I chose the map of San Francisco, California.
The py file contains the code, and the html and pdf files contain the whole report.

Summary
• Chose a region and used data munging techniques to assess the quality of the data for validity, accuracy 
• Analyzed data from osm file of 961 MB, 4581032 nodes, 532313 ways and 2566 unique users 
• Findings: The three most important source of information are City of Redwood, Tiger GPS, and Bing
